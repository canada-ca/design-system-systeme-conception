Recommended templates for Canada.ca

From Treasury Board of Canada Secretariat

These templates are recommended templates. They have been user-tested, and have been designed to help people complete their tasks.

Consider them as groups of design patterns. Don't hesitate to adapt to your needs and be creative.

Always focus on the task: do the design choices help people successfully complete the task they are trying to accomplish?

Recommended pattern groups

Template name Purpose Usage Type Working example
Service initiation page
  • Helps people quickly start a task
  • Gives people the information they need before starting the task, such as explaining eligibility criteria or steps to follow
  • Gives people easy access to:
    • service standards
    • turnaround times
    • related services
    • guides
    • help
  • Use to start tasks that:
    • have prerequisites to satisfy prior to starting the task, such as:
      • eligibility criteria
      • documents or information that must be in hand
    • point to any service delivery channel such as:
      • phone
      • email
      • mail
    • support services (as defined under the Policy on Service)
  • Don’t use for tasks that do not have any prerequisites. Use Generic destination pages instead
 Destination  
Program description page
  • Offers details about a program
  • Gives access to services related to the program
  • Use to give background information about a program
  • Don’t use to start a task
 Destination  
Collection landing page
  • Allows people to search within a collection
  • Introduces the most popular items from a collection
  • Gives access to an collection of common content produced by all or most federal institutions
  • Gives access to contextual and historical content related to a collection such as reports and events
  • Used only once to introduce a collection
 Navigation  
Consultation profile page
  • Describes a consultation or engagement activity
  • It can provide:
    • status and outcome of the initiative
    • background materials
    • data
    • information about planned or completed events
  • Use for each consultation, to provide an overview of a single consultation or engagement activity
 Navigation  
Act profile page
  • Gives a 1 or 2 sentence description written using plain language principles to describe the act
  • Allows people to read the legislation
  • Relates an act to:
    • topics
    • institutions
    • ministers
    • other legislation
  • Gives access to consultations on:
    • the act
    • news
    • notices
    • amendments
  • Use to publish an act
 Navigation Working example: act profile page
Regulation profile page
  • Gives a 1 or 2 sentence description written using plain language principles to describe the regulation
  • Allows people to read the regulation
  • Relates a regulation to:
    • topics
    • institutions
    • ministers
    • other legislation
  • Gives access to consultations on:
    • the regulation
    • news
    • notices
    • amendments
  • Use to publish a regulation
 Navigation Working example: regulation profile page
Guidance on legislation profile page
  • Gives access to guidance on legislation
  • Recaps the guidance using plain language principles
  • Gives access to details and related materials
  • Use to publish and explain guidance on legislation
 Navigation  
Campaign page
  • Gives information to support a promotional campaign
  • Allows for flexible page design
  • Use to support institutional, theme or GC-wide promotional campaigns that last for a set period of time
  • Don’t use as a replacement for other destination page types such as:
    • program delivery
    • service delivery
 Navigation  
Promotional events page
  • Describes an official government event that occurs in a promotional context
  • Gives information such as:
    • date
    • time
    • location
    • target audiences
  • May link to content related to the event such as:
    • services
    • programs, etc.
  • Use for an event that occurs that is part of a promotional campaign, such as a:
    • festival
    • celebration
    • ceremony
  • Don’t use for non-promotional events that occur as part of regular business activities. Non-promotional events may include:
    • workshops
    • webinars
 Navigation  
Short index page
  • Shows a combined list of select hyperlinked titles that lead to other pages or documents
  • Use to present a list of 10 to 50 related links without text descriptions
 Navigation  
Long index page
  • Shows a combined list of select hyperlinked titles that lead to other pages or documents
  • Use to present a list of 50+ related links without descriptions
 Navigation  
Finder page
  • Shows a combined list of select hyperlinked titles that lead to other pages
  • Lets people select only one value per metadata field in order to reduce a large amount of results of items into a shorter list
  • Metadata fields help people understand how a collection of items is organized
  • Use to present a list of 100+ related links that can be filtered using metadata and may require results to be split across multiple pages
  • Use when there is a need to allow people to build simpler queries against large amount of results
 Navigation  
Faceted finder page
  • Presents a combined list of select hyperlinked titles that lead to other pages or documents
  • Lets people select 1 or more values per metadata field in order to reduce a large collection of items into a more manageable list
  • Metadata fields help people understand how a large amount of results is organized
  • Use to present a list of 100+ related links that can be filtered using metadata and may require results to be split across multiple pages
  • Use when there is a need to allow people to build more complex queries against a large number of results
 Navigation
