H1 subtitle: Alpha design pattern

Some pages have a subtitle that describes a category the page belongs to. Render that subtitle in a way that more clearly shows it's part of that category, making the title easier to read.

When to use

Many pages fit into a section or category of other pages. To make the context of the page clear, we often make that section or category part of the title. For example:

How to apply - EI maternity and parental benefits

This is important because even though we want to focus on "How to apply", we need it to be clear that maternity and parental benefits are what are being applied to.

Use when you would write the main page heading with a subtitle that describes the category or section the page belongs to.

Implementation

A single h1 is used with spans to separate the 3 components:

  • the page name, which should be read 1st
  • the separator, which should be a colon
  • the subtitle, which should be read last

Use the following HTML for the h1:

<h1 property="name" id="wb-cont" dir="ltr">
  <span class="stacked" role="text">
    <span>Page name</span>: <span>Subtitle</span>
  </span>
</h1>
